Example 2:
Simplify the following linear algebraic expressions:
a) x + 2x − 5x
b) −3y + 6y − 2y
c) 3a − 2a − a
Solution 2:
a)
x + 2x − 5x
All are like terms in variable x.
= (1 + 2 − 5)x = −2x

b)
−3y + 6y − 2y
All are like terms in variable y.
= (−3 + 6 − 2)y = 1y = y

c)
3a − 2a − a
All are like terms in variable a.
= (3 − 2 − 1)a = 0a = 0
