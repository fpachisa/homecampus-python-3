3.2.2 Rules for Determining Significant Figures
Rule 1: All non-zero digits are significant.
Example 5:
State the number of significant figures in each number:
a) 3681   b) 4.6   c) 46.215

Solution 5:
a) The number 3681 has 4 significant figures.
b) The number 4.6 has 2 significant figures.
c) The number 46.215 has 5 significant figures.
Rule 2: All zeros between non-zero digits are significant.
Example 6:
State the number of significant figures in each number:
a) 1006   b) 20.078   c) 90.803
