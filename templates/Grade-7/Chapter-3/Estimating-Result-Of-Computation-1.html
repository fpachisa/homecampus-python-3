3.3.2 Estimating the Results of Computation
Sometimes, while using a calculator, we may key in a wrong value and get an incorrect answer. Thus, we use estimation to check if the answers are reasonable.
Example 16:
A student evaluates 39.5 + 13.2 − 10.1 using a calculator and gets 128.9 as the answer. Without doing actual calculation, use estimation to determine whether the answer is reasonable.

Solution 16:
39.5 + 13.2 − 10.1
≈ 40 + 13 − 10

= 43
∴ The student's answer is wrong.
Using a calculator, the actual answer is 42.6
Hence, the estimated value 43 is close to the actual value 42.6.
