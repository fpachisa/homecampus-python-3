10.1.2 Representation of Inequalities on Number Line

Solutions of an inequality can be represented on a number line as follows:

x > −4

Some of the values of x are −3, −2, −0.8, 0, 1, 2, 53 and so on. Note that x can be any real number with a value greater than −4.

The hollow circle ○ indicates that x is greater than but not equal to −4, i.e. the number −4 is not included.

Let us consider another example: x ≤ 2

Some of the values of x are 2, 1, 0.5, −1, −1.9, and so on.

The solid circle ● indicates that x is less than or equal to 2 i.e. the number 2 is included.
