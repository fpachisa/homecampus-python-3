Example 13:
A summer camp 'Lovely Summers' is organized for school children every year. It takes in 100 children every year. The manager of the summer camp splits these children into z groups, each group having the same number of children.

a)
Find the number of children in each group.
b)
Let the number of children in each group be y. Express the relation between y and z using a formula.
c)
Given that every year 10 groups are formed, find the number of children in each group using the formula derived in part (b).
Solution 13:
a)
Total number of children = 100

Number of Groups = z

∴ Number of children in each group =

100
___

z

b)
Using (a), we get the formula, y =

100
___

z
