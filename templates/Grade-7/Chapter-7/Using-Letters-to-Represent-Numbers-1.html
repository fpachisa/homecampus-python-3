7.1 Using Letters to represent numbers
The word algebra is derived from an Arabic word al-jebr that means "reunion of broken parts".
Example 1:
__ − 5 = 2. Find a number to fill in the blank.
We know that 7 − 5 = 2.
Therefore, the answer is 7.
In Algebra, we do not use blanks. Instead, we use letters to represent unknown numbers. So, the above question could be written as:
y − 5 = 2. Find y.
Any letter can be used to represent an unknown number. Such a letter is called a variable. If there are more than one unknown letters, we can use a different letter for each one. For example, a, b, p, q, x, y, z, etc. can all be used as variables.
In the above example, y − 5 is called an algebraic expression.
