5.3 Percentage Increase and Decrease
5.3.1 Percentage Increase
Let us study the following example to understand this concept.
Example 15:
It is forecasted that the current world population is 7.2 billion. It is projected to reach 9.6 billion by 2050. Find the percentage increase in the world population size?

Solution 15:
Increase in population
  =  
(9.6 − 7.2) billion

  =  
2.4 billion
The world population will increase by 2.4 billion over and above the current size of 7.2 billion. Thus, we calculate the percentage increase over initial value as:
Percentage increase
=

2.4 billion
________

7.2 billion

× 100 %

=
33.33 %
