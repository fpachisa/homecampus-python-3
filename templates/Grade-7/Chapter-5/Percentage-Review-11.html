5.1.4 Percentages Greater Than 100%
Example 11:
The male:female sex ratio in Singapore in 2013 was 968:1000.
a) Express this ratio in percentage value.
b) Find the number of females as a percentage of the number of males.

Solution 11:
a)
Required percentage
=

968
____

1000

× 100 %

= 96.8 %

Note: This means that the number of males is 3.2% (100% − 96.8%) lower than the number of females.
b)
Required percentage
=

1000
____

968

× 100 %

= 103.3 %

Note: The percentage value above 100% means that the number of females is 3.3 % (103.3% − 100%) higher than the number of males.
