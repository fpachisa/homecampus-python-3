Subtracting Like and Unlike Proper Fractions

Practice Unlimited Questions

How to subtract like fractions?

Step 1: Subtract all the numerators. (Denominator will remain the same.)
Step 2: Simplify if required.

See the example below
Subtract Fractions Calculator

How to subtract unlike fractions?

Step 1: Change unlike fractions to like fractions. i.e. find a common denominator.
Step 2: Subtract all the numerators.
Step 3: Simplify if required.

See the example below

2. Monica had   {{ frGen(1,4) }} of a pie and Sarah had   {{ frGen(3,4) }} of the pie.
How much more of the pie had Sarah than Monica?

Sarah  =>   {{ frGen(3,4) }}
Monica  =>   {{ frGen(1,4) }}
Sarah  −  Monica  =>   {{ frGen(3,4) }}  −  {{ frGen(1,4) }}

    {{ frGen(3,4) }}  −  {{ frGen(1,4) }}
=  {{ frGen("3 - 1",4) }}
=  {{ frGen(2,4) }}
=  {{ frGen("2 / 2","4 / 2") }} (simplify)
=  {{ frGen(1,2) }}
Sarah had {{ frGen(1,2) }} a pie more than Monica.

3. Elly gives   {{ frGen(1,2) }} of a pizza to Charlie and   {{ frGen(1,3) }} of the pizza to Derek.
How much more of the pizza does Elly give to Charlie than Derek?

Charlie  =>   {{ frGen(1,2) }}
Derek  =>   {{ frGen(1,3) }}
Charlie  −  Derek  =>   {{ frGen(1,2) }}  −  {{ frGen(1,3) }}


Both Charlie and Derek had 1 slice each but Charlie's slice was bigger than Derek's slice because Charlie's pizza was cut into 2 while Derek's pizza was cut into 3.

To compare the two slices, first cut both pizzas into equal number of slices.
    {{ frGen(1,2) }}  −  {{ frGen(1,3) }}
=  {{ frGen("1 x 3","2 x 3") }}  −  {{ frGen("1 x 2","3 x 2") }}
=  {{ frGen(3,6) }}  −  {{ frGen(2,6) }}   (changed the denominator to 6)
=  {{ frGen(1,6) }}

Elly gives {{ frGen(1,6) }} of a pizza more to Charlie than to Derek.
Since the two fractions, {{ frGen(1,2) }} and {{ frGen(1,3) }} are unlike fractions (as the denominators are different), we must first change them to like fractions before we can subtract them (that is, find a common denominator).
