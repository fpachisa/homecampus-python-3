What is a mixed number?

A mixed number or mixed fraction has two parts: a whole number and a proper fraction.

Mixed number  =  whole number  +  proper fraction

    2. How many pizzas do I have altogether if I have  {{ mixNumGen(1,2,3) }}  pepperoni pizzas and  {{ mixNumGen(2,1,2) }}  cheese pizzas?

Pepperoni pizza  =>   {{ mixNumGen(1,2,3) }}
Cheese pizza  =>   {{ mixNumGen(2,1,2) }}
Total  =>   {{ mixNumGen(1,2,3) }}  +  {{ mixNumGen(2,1,2) }}



      {{ mixNumGen(1,2,3) }}  +  {{ mixNumGen(2,1,2) }}
=  {{ frGen("(1 x 3) + 2",3) }}  +  {{ frGen("(2 x 2) + 1",2) }}    (First, change both mixed numbers to improper fractions)
=  {{ frGen(5,3) }}  +  {{ frGen(5,2) }}
=  {{ frGen("5 x 2","3 x 2") }}  +  {{ frGen("5 x 3","2 x 3") }}    (Second, change the fractions to like fractions)
=  {{ frGen(10,6) }}  +  {{ frGen(15,6) }}
=  {{ frGen(25,6) }}    (Next, add the two fractions)
=  4  +  {{ frGen(1,6) }}    (Finally, change the improper fraction to a mixed number)
=  {{ mixNumGen(4,1,6) }}
I have   {{ mixNumGen(4,1,6) }}   pizzas altogether.
