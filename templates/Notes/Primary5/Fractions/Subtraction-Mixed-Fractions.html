1. The video below explains how to subtract mixed numbers.

A mixed number or mixed fraction has two parts: a whole number and a proper fraction.

Mixed number  =  whole number  +  proper fraction

    2. I have {{ mixNumGen(2,5,6) }} pepperoni pizzas and {{ mixNumGen(1,2,3) }} cheese pizzas. How many more pepperoni pizzas do I have than cheese pizzas?

Pepperoni pizzas  =>   {{ mixNumGen(2,5,6) }}
Cheese pizzas  =>   {{ mixNumGen(1,2,3) }}
Pepperoni  −  Cheese  =>   {{ mixNumGen(2,5,6) }}  −  {{ mixNumGen(1,2,3) }}  =  ?

    {{ mixNumGen(2,5,6) }}  −  {{ mixNumGen(1,2,3) }}
=  {{ frGen("(2 x 6) + 5",6) }}  −  {{ frGen("(1 x 3) + 2",3) }}    (First, change both mixed numbers to improper fractions)
=  {{ frGen(17,6) }}  −  {{ frGen(5,3) }}
=  {{ frGen("17 x 1","6 x 1") }}  −  {{ frGen("5 x 2","3 x 2") }}    (Second, change the fractions to like fractions)
=  {{ frGen(17,6) }}  −  {{ frGen(10,6) }}    (Next, subtract the fractions)
=  {{ frGen(7,6) }}
=  1 + {{ frGen(1,6) }}    (Finally, change the improper fraction to mixed number)
=  {{ mixNumGen(1,1,6) }}
I have  {{ mixNumGen(1,1,6) }}  more pepperoni pizzas than cheese pizzas.
