    2. {{ frGen(3,4) }} of a pizza is divided equally among 3 friends. What is the share of each friend?

    {{ frGen(3,4) }}  ÷  3
=  {{ frGen(3,4) }}  ÷  {{ frGen(3,1) }}
=  {{ frSimplifyNumGen(3,4,1) }}  ×  {{ frSimplifyDenGen(1,3,1) }}
=  {{ frGen("1 x 1","4 x 1") }}
=  {{ frGen(1,4) }}

Each friend receives {{ frGen(1,4) }} of the pizza.
Dividing by 3 or {{ frGen(3,1) }} is the same as multiplying by {{ frGen(1,3) }} .
