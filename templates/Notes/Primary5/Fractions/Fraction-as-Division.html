    2. Show that 3 divided by 4 is {{ frGen(3,4) }}.

Let's suppose we have 3 pizzas that are divided equally among 4 friends  =>  3  ÷  4.
We will show that each friend gets  =>  {{ frGen(3,4) }} of a pizza.


      3  ÷  4
=  {{ frGen(1,4) }}   of   3 (pizzas)
=  {{ frGen(1,4) }}  ×  3
=  {{ frGen(1,4) }}  ×  {{ frGen(3,1) }}
=  {{ frGen("1 x 3","4 x 1") }}
=  {{ frGen(3,4) }}
3 pizzas are divided among 4 friends.

So, each of the friend gets {{ frGen(1,4) }} of each of the 3 pizzas.
Therefore, each friend gets {{ frGen(3,4) }} of the pizza.
