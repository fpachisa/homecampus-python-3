What is Speed?

Speed describes how fast an object is moving. In other words, speed tells us the distance an object travels per unit of time.

The various units of speed are:
  • cm/s (read as, centimetres per second)
  • m/min (read as, metres per minute)
  • m/s (read as, metres per second)
  • km/h (read as, kilometres per hour)
