How to divide a proper fraction by a proper fraction?

Dividing a fraction by another fraction is the same as multiplying the first fraction by the inverse of the second fraction. Here are the steps:

Step 1:   Change  ÷  to  × .
Step 2:   Invert the second fraction.
Step 3:   Simplify the expression.

Example:
Divide {{ frGen(4,9) }} by {{ frGen(2,3) }}.
{{ frGen(4,9) }}  ÷  {{ frGen(2,3) }}  =  {{ frGen(4,9) }}  ×  {{ frGen(3,2) }}  =  {{ frGen(2,3) }}

5. Maya had     {{ frGen(3,4) }} kg of rice.
She packed the rice into bags of     {{ frGen(1,10) }} kg each.
a)   How many bags did she make?
b)   How much rice was left unpacked?
c)   How much more rice does she need to make another bag of     {{ frGen(1,10) }} kg?


a)     {{ frGen(3,4) }}  ÷  {{ frGen(1,10) }}
    =  {{ frGen(3,4) }}  ×  {{ frGen(10,1) }}
    =  {{ frGen(30,4) }}
    =  {{ mixNumGen(6,1,2) }}

    She made 6 bags.

b)     6  ×  {{ frGen(1,10) }}  =  {{ frGen(3,5) }}

    There was {{ frGen(3,5) }} kg of rice in 6 bags altogether.

    {{ frGen(3,4) }}  −  {{ frGen(3,5) }}  =  {{ frGen(1,20) }}
    There was {{ frGen(1,20) }} kg rice that was left unpacked.

c)
    {{ frGen(1,10) }}  −  {{ frGen(1,20) }}  =  {{ frGen(1,20) }}
    She needs {{ frGen(1,20) }} kg more rice to make another bag of {{ frGen(1,10) }} kg.

6. A rectangular piece of land     {{ frGen(1,3) }} km long has an area of     {{ frGen(1,12) }} km2.
Find the length of fence needed to border the land.


    Area of the rectangular land  =  length  ×  breadth
     {{ frGen(1,12) }} km2  =  {{ frGen(1,3) }} km  ×  breadth

      Breadth
=  {{ frGen(1,12) }}  ÷  {{ frGen(1,3) }}
=  {{ frGen(1,12) }}  ×  {{ frGen(3,1) }}
=  {{ frGen(1,4) }} km

     Border or Perimeter
=  (2  ×  length)  +  (2  ×  breadth)
=  (2  ×  {{ frGen(1,3) }})  +  (2  ×  {{ frGen(1,4) }})
=  {{ frGen(2,3) }}  +  {{ frGen(1,2) }}
=  {{ frGen(7,6) }} km
=  {{ mixNumGen(1,1,6) }} km

The length of fence needed to border the land is   {{ mixNumGen(1,1,6) }}   km.
